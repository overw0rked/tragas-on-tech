## 📐 El espacio · Patio del rDl

El Patio del rDl mide aproximadamente **30 m²**, lo que fija una **capacidad
máxima de 12 personas**. Esto considera **un par de mesas plegables**:

- 🍳 una como **área de cocina** (asador + comal)
- 🔪 otra como **área de preparación**

Esta guía **no contempla una mesa común**: se asume que parte de la experiencia de
la carnita asada es **desplazarse a través del espacio para generar convivencia**.

```
        Patio del rDl · ~30 m² · aforo 12
   ┌───────────────────────────────────────┐
   │  [ Asador + Comal ]                    │
   │  ▭▭ Mesa «Cocina»                      │
   │                                        │
   │              ←  convivencia  →         │
   │            (el espacio se camina)      │
   │                                        │
   │  ▭▭ Mesa «Preparación»            🔪   │
   └───────────────────────────────────────┘
```

## ⚡ Lo esencial de un vistazo

- 👥 **Aforo:** 12 personas
- 🥩 **Carne:** ~250 g por persona (≈ **3 kg** para aforo lleno)
- 🍽️ **Estructura:** corte botanero de entrada → platillo principal → guarniciones asadas
- 🌙 **Evento nocturno:** iluminación, repelente y extensiones de color brillante
- 🏅 **Cada aportación en especie** suma al [Cuadro de Honor](CUADRO-DE-HONOR.md)
